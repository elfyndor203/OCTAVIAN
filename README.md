# OCTAVIAN engine (WORK IN PROGRESS)
A simple modular ECS game engine written in C 

# Libraries
- OpenGL 3.3
- GLAD
- GLFW

## Installation (WIP)
1. Build
2. Import .lib

## Usage (WIP)
1. Start the engine
2. Create an entityContext
3. Add the entityContext to rendering
4. Add an entity to the entityContext
5. Add any additional components to the entity
7. Handle messages
